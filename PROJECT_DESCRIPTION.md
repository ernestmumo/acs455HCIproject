# Tradeflow HCI Project

## Description
The Tradeflow HCI project focuses on creating an intuitive and user-friendly interface for managing trade data. It aims to streamline the process of data entry, manipulation, and visualization for users involved in trading activities.

## Technology Stack
- **Front-end:** React.js
- **Back-end:** Node.js with Express
- **Database:** MongoDB
- **Hosting:** AWS
- **Styling:** Tailwind CSS

## Composition
1. **User Dashboard:** A dashboard providing insights into trading activities and data visualizations.
2. **Data Entry Forms:** Forms for users to input and manage their trading data efficiently.
3. **Reports Section:** Generating reports based on user data with options to filter and export.
4. **User Authentication:** Secure login and registration processes to protect user data.
5. **API Integration:** Connecting to external trading platforms for real-time data.

This file serves as an overview of the Tradeflow HCI project and its main components.
